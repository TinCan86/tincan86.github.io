Övning 16 - React Events

Fortsättning på 12


Denna uppgift bygger vidare på uppgift 12. Observera att det är förbjudet att använda addEventListener, alla events ska hanteras med React. Din uppgift är att rendera en React-komponent som gör ett AJAX request och hämta en lista med JSON objekt. Börja med att välja ut ett lämpligt API. (Det är ok att använda landsinformationen på http://forverkliga.se/JavaScript/api/simple.php?world ) AJAX ska göras när komponenten renderats första gången. Innan något svar kommit ska det stå ett lämpligt meddelande på sidan.

När svaret kommer ska det presenteras i en lista eller en tabell. Efter listan ska det stå hur många element som finns i den. Gör en eller flera komponenter för att visa informationen i ett JSON-objekt. Välj ut den viktigaste informationen. Använd HTML och CSS för att göra informationen lättläst. Det är inte ok att skriva ut en JSON-sträng direkt.

Användaren ska kunna klicka på ett element i listan för att markera det. Då ska det dyka upp ett sätt som användaren kan ta bort elementet. (Tips: använd conditional rendering.)

För VG:
1 När användaren klickar på något inuti elementet så ska texten ändras till ett textfält, så att användaren kan ändra texten i listan direkt.
2 Lägg till ett textfält. När användaren skriver text i det ska listan filtreras, så att bara de element som innehåller texten i textfältet ska renderas.

Några användbara events: onClick, onChange, onBlur, onKeyPress
