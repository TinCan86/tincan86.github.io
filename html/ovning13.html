Övning 13 - React

Element, component, props, state


Skapa en HTML-sida som man kan köra React på. Du kommer behöva det för inlämningen. Lös uppgifterna antingen på den sidan eller med JS Bin eller Code pen. Läs dokumentationen på https://facebook.github.io/react/docs/hello-world.html allteftersom du går igenom övningarna.

  1. Skapa ett React element och rendera det. Elementet kan skriva ut vad som helst på webbsidan.

  2. Gör ett React element som lägger ihop talen 333 och 777 och skriver ut vad summan blir.

  3. Gör ett React element som innehåller flera HTML element.

  4. Gör ett React element som använder props för att skriva ut något.
